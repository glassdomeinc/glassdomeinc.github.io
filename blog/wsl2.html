WSL2: Windows Subsystem for Linux 2​

WSL stands for "Windows Subsystem for Linux", a tool that makes Linux available to Windows.

Before WSL came out, we used [Cygwin] (https://www.cygwin.com/)) or a method of installing a Linux distribution (such as Ubuntu) as a VM on a PC. Cygwin is not a complete Linux implementation, however, and installing Linux as a VM is costly. Instead, WSL makes it convenient for Windows to use Linux.

Advantages of WSL​

WSL2 allows Windows PC users to have the same user experience as they do with the Darwin operating system. Installing and using WSL has the following advantages:

  • You can easily install the tools you need by using a package manager such as homebrew.
  • In many cases, you can quickly solve any problems using linux shell commands

Above all, when documenting any troubleshooting method, Windows users and macOS users do not have to write separate solutions which optimizes the team's communication and development productivity.

Disadvantages of WSL​

WSL is different from the actual Linux distribution and cannot be used for servers. (systemd not supported)

Instead, there is virtually no downside for us to use PCs.

How to install​

Open PowerShell with administrator privileges

wsl --install

This will install (based on Windows 11)

  • [한국어] WSL을 사용하여 Windows에 Linux 설치

More: Windows Terminal​

It is better to run the shell using Windows Terminal provided by Windows than to run the WSL distribution directly (for example, Ubuntu).

Use shortcuts, such as copy-paste, which are more compatible with other programs, and the interface is convenient.
